# WearGest

WearGest is a gesture-based mouse control app for Android Wear devices. It allows users to control their mouse using gestures on their wearable device.

## Features

- Gesture-based mouse control
- Customizable gestures
- Support for multiple devices
- Easy to use and setup

## Installation

1. Download the app from the Google Play Store
2. Install the app on your Android Wear device
3. Follow the on-screen instructions to complete the setup

## Usage

1. Open the app on your Android Wear device
2. Follow the on-screen instructions to complete the setup
3. Use the gestures to control your mouse
